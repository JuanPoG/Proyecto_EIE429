Bienvenido al Proyecto IoT

Aquí puedes encontrar información sobre nuestro proyecto de Internet de las Cosas.

Descripción del Proyecto

El proyecto IoT tiene como objetivo principal la monitorización de variables ambientales como la temperatura, la humedad y la calidad del aire utilizando sensores avanzados. Nuestro sistema permite la visualización en tiempo real de estos datos a través de gráficos interactivos, facilitando así la toma de decisiones informadas sobre el entorno.

Sensores Utilizados

Sensor de Calidad del Aire MQ-135: El sensor MQ-135 se utiliza para medir la calidad del aire, detectando gases nocivos como el dióxido de carbono (CO2), el amoníaco (NH3), el alcohol, el benceno, el humo y otros contaminantes presentes en el aire. Este sensor es altamente sensible y proporciona datos precisos sobre la concentración de estos gases, permitiendo una monitorización eficiente de la calidad del aire en diferentes entornos.

Sensor de Temperatura y Humedad DHT22: El sensor DHT22 es un sensor de bajo costo y alta precisión que mide tanto la temperatura como la humedad del ambiente. Utiliza un sensor capacitivo de humedad y un termistor para medir el aire circundante, proporcionando lecturas digitales precisas. Este sensor es ideal para aplicaciones de monitoreo ambiental donde se requiere una alta precisión y estabilidad a largo plazo.
